这是一个 HTML 网页版「死了吗」App 复刻版(瞎玩的，非商业版)。

这个版本高度还原了此类 App 的核心逻辑：极简主义设计、黑色幽默风格、倒计时机制以及「续命」操作。

特点：
1.极简暗黑风：纯黑背景，高对比度文字，营造严肃又带有赛博朋克感的氛围。 2. 数据持久化：使用了浏览器本地存储 (localStorage)，即使你关闭浏览器或刷新页面，倒计时也不会重置，除非时间耗尽或你手动点击续命。
2.高度可配置：你可以点击右上角设置，自定义「倒计时时长」和「遗言」。
3.模拟触发：当倒计时归零，会变红并模拟发送遗言的状态。


核心功能逻辑说明：
倒计时逻辑：

不像普通的闹钟，这个倒计时是基于时间戳 (Timestamp) 的。当你点击「一键续命」时，它会计算出未来的一个时间点（比如现在是 12:00，周期 24 小时，截止时间就是明天的 12:00）。
这意味着：哪怕你关掉网页、关机、明天再打开网页，倒计时依然是准确的（会显示剩余时间或已超时）。
视觉反馈：
正常状态：白色数字。
危险状态：当剩余时间不足 1 小时，数字会变红并开始闪烁（呼吸灯效果）。
死亡状态：全屏红色闪烁遮罩，提示「已触发紧急联系人」（这是模拟效果，纯 Web 无法在后台发短信）。
防误触与复活：
如果超时了，界面会锁定。此时提供了一个「误报：点击解除警报」的按钮，模拟现实中如果用户只是睡过头了，可以手动解除警报并重置周期。
